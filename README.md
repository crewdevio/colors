# Colors terminal a tiny library to print colors for console in dart

Colors terminal use ansi code to print colors

### 8 bits colors

- red
- black
- green
- yellow
- blue
- magenta
- cyan
- white

### 16 bits colors

- brightBlack
- brightRed
- brightGreen
- brightYellow
- brightBlue
- brightMagenta
- brightCyan
- brightWhite

#### Background Colors

- backgroundBlack
- backgroundRed
- backgroundGreen
- backgroundYellow
- backgroundBlue
- backgroundMagenta
- backgroundCyan
- backgroundWhite

- backgroundBrightBlack
- backgroundBrightRed
- backgroundBrightGreen
- backgroundBrightYellow
- backgroundBrightBlue
- backgroundBrightMagenta
- backgroundBrightCyan
- backgroundBrightWhite

### text decoration

- bold
- underline
- reversed
